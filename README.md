## Deployment

Each subdirectory can be deployed as a separate Vercel project:

1. Create a new Vercel project
2. Set the root directory to the specific domain folder (e.g., `xr-studios.com`)
3. Deploy

Or deploy all at once using Vercel's monorepo support.
